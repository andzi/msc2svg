
msc2svg version @version@
http://msc2svg.google.com

1) Requirements:
 Sun Java 1.5 or higher (http://www.java.com/download/)
2) Installation
 2.1) Microsoft Windows:
      2.1.1) Unzip msc2svg-@version@-bin.zip 
	  2.1.2) Open run.cmd in your favorite editor and edit the first line 
	         to set JAVA_HOME to point to your java  installation directory
	         (For example : set JAVA_HOME="c:\Program Files\Java\jdk1.6.0_10")

3) Execution
      3.2.1) Open a command line window and execute run.cmd or just click on run.cmd.
